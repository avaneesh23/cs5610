Strengths of BackboneJS over Knock-out, Angular and Ember JS

  • Built faster, light weight, modular, highly customizable, MVC based SPA (Single Page App) using backboneJS
  • Implemented mouse and click actions on the flower app using backbone's events feature.
  • Decided to use backboneJS with nodejs for my final project over angular or ember.
  • Used underscore library along with jquery's functions such as html(), addClass(), removeClass to implement backbone SPA.
