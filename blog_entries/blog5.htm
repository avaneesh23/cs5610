Restaurant app using Couch DB on the cloud

  • I built a simple Restaurant application using couch DB that retrieves the images of different menu items from the database.
  • All resources on the database including images can be accessed from the URL. Implemented map functions to retrieve data from NoSQL database.
  • All NO SQL databases are listed as a JSON response at http://cs5610.iriscouch.com/_all_dbs/
  • Installed all the dependencies by configuring them in package.json file.
  • Used "cradle" library to connect my application to Couch DB. Bundled curl along with the application.
